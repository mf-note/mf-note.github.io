## 开发环境要求

* Node版本 >= 12.2.0
* npm版本 >= 6

## 初始化一个 Svelte 项目

### 官方脚手架

* 安装 `degit`
  `npm install -g degit`

* 使用 `degit` 创建项目
  `degit sveltejs/template your-project`

* 安装项目依赖
  ```
  cd your-project
  npm install
  npm run dev
  ```

> `degit` 创建的项目默认使用 `rollup` 打包和编译
